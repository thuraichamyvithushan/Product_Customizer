# Color Update Instructions

The colors have been updated in the code. To see the changes:

1. **Stop the dev server** (if running) - Press Ctrl+C
2. **Restart the dev server:**
   ```bash
   cd frontend
   npm run dev
   ```

3. **Clear browser cache:**
   - Press Ctrl+Shift+R (Windows/Linux) or Cmd+Shift+R (Mac) for hard refresh
   - Or open DevTools (F12) → Right-click refresh button → "Empty Cache and Hard Reload"

## New Colors:
- **Main Color (Brand)**: #0a214f (Dark Blue)
- **Buttons/Accent**: #fe7245 (Orange/Coral)

All buttons now use the orange accent color (#fe7245).
Text links and headings use the dark blue brand color (#0a214f).
